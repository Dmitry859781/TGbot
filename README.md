# TGshkaBot

Данный бот создан для ведения заметок в телеграме

## Основные функции:
1) Обычные заметки
2) Оповещения, по времени со следующими свойствами:
  2.1) по определённым дням
  2.2) по определённому времени
  2.3) через определённый промежуток времени
  2.4) повторные оповещения(через каждые 5 минут)
3) Реализовать группы заметок/оповещений
4) Отключение всех оповещений выбранной группы

## Список задач

Задача 1: 
Цель: Сделать заготовку чат-бота для Telegram, которая умеет отвечать на несколько команд:
- /about — выводит краткую информацию про бота
- /authors — выводит информацию об авторах
- /help — выводит информацию обо всех доступных командах (можно запускать с аргументом: /help <command>.
