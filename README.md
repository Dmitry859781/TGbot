## Zametochki Telegram Bot

Zametochki - Телеграм бот, который предоставляет набор полезных инструментов прямо в чате: управление заметками, гибкие напоминания с поддержкой часовых поясов и другие функции.

### Основные возможности добавленные в текущей версии

Показ информации о боте, авторах и вызов справки

### Для запуска бота необходимы:

Java 17 или выше

Maven

Токен Telegram-бота, полученный от BotFather

## Команды бота

Основные команды, доступные пользователям:

/about — Показать информацию о боте

/author — Показать информацию об авторах

/help - Вывод справки

### Архитектура

Проект реализован на Java 17 с использованием фреймворка TelegramBots для взаимодействия с Telegram API. В качестве базы данных используется SQLite, что исключает необходимость в дополнительных серверах СУБД. Сборка проекта осуществляется через Maven, а тестирование — с помощью JUnit 5 и Mockito.
